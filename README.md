# 📊 Sales Analytics Platform

Aplikasi analisis data penjualan berbasis Python dan Streamlit untuk membantu bisnis memahami performa penjualan, pola pembelian pelanggan, serta menghasilkan laporan eksekutif secara otomatis.

## 🚀 Live Demo

🔗 https://toleransi-kopi-analytics.streamlit.app

## 📸 Preview

(Tambahkan screenshot dashboard di sini)

## ✨ Fitur Utama

- Upload data CSV / Excel
- Data Cleaning Otomatis
- Dashboard KPI Penjualan
- Visualisasi Data Interaktif
- Market Basket Analysis (Apriori)
- Forecast Penjualan
- Customer Segmentation
- Executive Report PDF
- Executive Presentation PPT
- Profile Management

## 📈 Dashboard KPI

Menampilkan:

- Total Revenue
- Total Transaksi
- Total Item Terjual
- Average Order Value (AOV)
- Top Product
- Revenue by Category
- Revenue by Hour
- Revenue by Staff

## 🛒 Market Basket Analysis

Menggunakan algoritma Apriori untuk menemukan pola pembelian pelanggan berdasarkan data transaksi.

Contoh insight:

> Pelanggan yang membeli Americano cenderung membeli Croissant.

## 🔮 Forecast Penjualan

Memprediksi tren penjualan di masa mendatang menggunakan metode forecasting.

## 📄 Executive Report

Generate laporan bisnis otomatis dalam format:

- PDF Executive Report
- PowerPoint Executive Presentation

## ⚙️ Instalasi

Clone repository:

```bash
git clone https://github.com/zenithsky73/sales-analytics-app.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run aplikasi:

```bash
streamlit run app.py
```
